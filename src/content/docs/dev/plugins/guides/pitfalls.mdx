## 1: Avoid using browser APIs

Plugins should **avoid** interacting with the browser APIs and use the API provided by Limbo **as much as possible**. A mobile version of Limbo is on the [Roadmap](roadmap) and plugins that use browser APIs may cause the plugin to be unusable.
